# 📊 HR Attrition Analysis (SQL Project)

## 📌 Project Overview
This project analyzes employee attrition using SQL. It helps identify patterns in employee exits, salary trends, attendance, and department performance.

---

## 🗂️ Database Structure
The project contains 3 tables:
- Employees
- Attrition
- Attendance

---

## 🔍 Key Analysis Performed
- Total Employees Count
- Department-wise Employee Distribution
- Attrition Rate Calculation
- Salary Analysis (Average & Above Average)
- Attrition by Department
- Absent Employee Analysis

---

## 🛠️ Tools Used
- MySQL
- SQL Queries

---

## 📈 Insights
- Identified departments with high attrition
- Found employees with high absenteeism
- Compared salaries across departments
