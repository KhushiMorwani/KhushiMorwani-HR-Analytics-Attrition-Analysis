
📊 HR Analytics & Attrition Analysis (SQL – Custom Dataset)
🔍 Objective

Analyze employee data to understand attrition patterns, salary trends, and workforce behavior.
This project simulates real-world HR analytics using a custom-built dataset.

🛠️ Tools Used

SQL (MySQL)

📁 Dataset

This project uses a custom-built HR dataset created using SQL.

The dataset includes:

Employee details (age, gender, department, salary, job role)

Attrition information (employee exit status and reasons)

Attendance records (presence/absence tracking)

The dataset was designed to simulate real-world HR business scenarios.

📊 Analysis Performed
🔹 Employee Analysis

Total number of employees

Department-wise employee distribution

🔹 Attrition Analysis

Attrition count

Attrition rate calculation

Attrition by department

🔹 Salary Analysis

Average salary by department

Employees earning above average salary

🔹 Attendance Analysis

Employees with highest absence days


📊 Key Insights

Certain departments show higher attrition rates

Employees with lower salaries tend to leave more frequently

High absence frequency may indicate potential attrition risk

Salary distribution varies significantly across departments

🚀 Conclusion

This project demonstrates how SQL can be used to analyze HR data, identify employee behavior patterns, and support data-driven decision-making in organizations.

⭐ Project Highlights

Custom dataset creation

Relational database design

Use of JOINs, GROUP BY, and subqueries

Business-focused insights
